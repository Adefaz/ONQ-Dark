Redirect
?

When you enter a course though course tiles on homepage or course selector on navigation bar, you will be automatically redirect to it's corresponding content page. You can still access the original course home through other links.

Welcome

Thanks for using Learn Darklight! This extension offers several themes for the new Waterloo Learn.

If you like this extension, please rate and leave a comment in Firefox Add-ons.

Notes

▷ Don't use other Waterloo Learn extensions with Darklight to avoid unexpected behaviors.

▷ Waterloo Learn may change at any time, thus the extension may not function properly.

▷ If you have any questions regarding this extension, please email me at sam.zj.shao@gmail.com or submit your feedback online.
